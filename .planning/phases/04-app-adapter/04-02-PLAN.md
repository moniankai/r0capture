**方法实现**：

1. `get_package_name() -> str`
   - 返回 `"com.phoenix.read"`（从 download_drama.py 的 APP_PACKAGE 常量）

2. `get_hook_script() -> str`
   - 返回 `"frida_hooks/ttengine_all.js"`

3. `parse_ui_context(xml: str, **kwargs) -> UIContext`
   - 委托给 `drama_download_common.parse_ui_context(xml)`
   - 导入：`from scripts.drama_download_common import parse_ui_context`
   - 直接返回现有实现的结果（保持向后兼容）

   - 委托给 `download_drama.select_episode_from_ui(ep_num)`
   - 需要从 download_drama 导入该函数（见 Task 2）
   - 传递 `max_attempts` 参数（从 kwargs 获取，默认 8）

**导入语句**：
- `from scripts.drama_download_common import parse_ui_context, UIContext`
- `from scripts.download_drama import select_episode_from_ui`（Task 2 后可用）

**设计要点**：
- 所有方法添加完整的文档字符串
- 使用类型注解
- 在 `get_hook_script` 文档中说明返回的是文件路径，不是脚本内容
