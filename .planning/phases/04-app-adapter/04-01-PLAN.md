  <action>
在 `scripts/app_adapter.py` 中添加工厂函数和 adapter 注册机制：

**工厂函数**：
```python
def create_adapter(app_name: str) -> AppAdapter:
    """根据 app_name 创建对应的 adapter 实例
    
    Args:
        app_name: App 标识符（如 'honguo', 'kuaishou', 'douyin'）
    
    Returns:
        AppAdapter 实例
    
    Raises:
        ValueError: 如果 app_name 不支持
    """
```

**注册机制**：
- 使用模块级字典 `_ADAPTER_REGISTRY: Dict[str, Type[AppAdapter]] = {}` 存储已注册的 adapter 类
- 提供装饰器 `@register_adapter(name: str)` 用于注册新 adapter
- `create_adapter` 从注册表中查找并实例化

**设计要点**：
- 工厂函数在 app_name 不存在时抛出 `ValueError`，错误信息包含可用的 adapter 列表
- 装饰器检查被装饰的类是否为 AppAdapter 子类，否则抛出 `TypeError`
- 添加辅助函数 `list_available_adapters() -> List[str]` 返回已注册的 adapter 名称列表

**示例用法**（在文档字符串中说明）：
```python
@register_adapter('honguo')
class HongGuoAdapter(AppAdapter):
    app_name = 'honguo'
    ...

adapter = create_adapter('honguo')  # 返回 HongGuoAdapter 实例
```
