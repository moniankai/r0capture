修改前（约第 1801 行）：
```python
def download_and_decrypt(ep_num: int):
    state = get_capture_state()
    
    # 直接使用第一个捕获的数据
    if not state.video_refs:
        logger.warning("[下载] 无 Hook 数据")
        return False
    
    vid_ref = state.video_refs[0]
    aes_key = state.aes_keys[0] if state.aes_keys else None
```

修改后：
```python
def download_and_decrypt(ep_num: int):
    state = get_capture_state()
    
    # 过滤最近 5 秒内的数据
    now = time.time()
    FRESHNESS_THRESHOLD = 5.0  # 秒
    
    recent_refs = [
        r for r in state.video_refs 
        if now - r.timestamp < FRESHNESS_THRESHOLD
    ]
    
    if not recent_refs:
        logger.warning(
            f"[下载] 无最近的 Hook 数据（{len(state.video_refs)} 个过期数据被丢弃），"
            "等待重新捕获..."
        )
        time.sleep(2)
        return False
    
    # 使用最新的数据（按时间戳排序，取最新）
    vid_ref = max(recent_refs, key=lambda r: r.timestamp)
    
    # 如果 AES 密钥也有时间戳，同样过滤
    if hasattr(state.aes_keys[0] if state.aes_keys else None, 'timestamp'):
        recent_keys = [
            k for k in state.aes_keys 
            if now - k.timestamp < FRESHNESS_THRESHOLD
        ]
        aes_key = max(recent_keys, key=lambda k: k.timestamp) if recent_keys else None
    else:
        # 向后兼容：如果 aes_keys 是字符串列表
        aes_key = state.aes_keys[0] if state.aes_keys else None
    
    logger.info(
        f"[下载] 使用 Hook 数据: video_id={vid_ref.video_id[:8]}, "
        f"age={now - vid_ref.timestamp:.1f}s"
    )
```

**关键点**：
- 定义 `FRESHNESS_THRESHOLD = 5.0` 常量（可调整）
- 使用列表推导式过滤最近 5 秒内的数据
- 如果无新鲜数据，记录警告并返回 `False`（触发重试）
- 使用 `max(..., key=lambda r: r.timestamp)` 选择最新数据（而非第一个）
- 添加日志显示数据年龄（`age=...s`）
- 保持向后兼容：如果 `aes_keys` 是字符串列表，使用 `hasattr` 检测
